# Project structure

## Directories :

docs - The directory contains project documentation in Russian and English.

image - The directory contains images used in the program.

resourse - directory contains . cs files describing the application’s functionality

## Models Directory :

Available files :

Achives.cs - contains the class Achives.

Level.cs - contains the class Levels.

Mouse.cs - contains the class Mouse.

Processor.cs - contains the class Processor.

SqlQuery.cs - contains the class SqlQuery.

Statistics.cs - contains the class Statistics.

StatsPlayer.cs - contains the StatsPlayer class.

VideoCard.cs - contains the VideoCard class.
